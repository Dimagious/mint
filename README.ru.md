**MINT = Merge Image'N Text** — лёгкий браузерный редактор для быстрого создания социальных картинок.

[![Node >= 18](https://img.shields.io/badge/node-%3E%3D18-339933?logo=node.js&logoColor=white)](https://nodejs.org/)
[![pnpm >= 9](https://img.shields.io/badge/pnpm-%3E%3D9-F69220?logo=pnpm&logoColor=white)](https://pnpm.io/)
[![Deploy: GitHub Pages](https://img.shields.io/badge/deploy-GitHub%20Pages-222?logo=github&logoColor=white)](./.github/workflows/deploy-pages.yml)

MINT помогает создавать визуалы для постов и сторис с текстовыми слоями, быстро экспортировать их и итерировать — без тяжёлых дизайн-инструментов.

## Название

**MINT** расшифровывается как **Merge Image'N Text** (слияние изображения и текста).

## Демо

Приложение доступно по адресу:

**[https://dimagious.github.io/mint/](https://dimagious.github.io/mint/)**

## Зачем это нужно

- Быстрые черновики контента для личных проектов.
- Чистый флоу: загрузить фон → добавить текстовые слои → экспортировать.
- Работает полностью в браузере — никакого сервера не нужно.

## Возможности

- Пресеты холста под популярные форматы соцсетей:
  - `1080 x 1080` Квадрат
  - `1080 x 1350` Портрет
  - `1080 x 1920` Сторис
- Загрузка фона перетаскиванием (drag-and-drop).
- Несколько текстовых слоёв с настройками:
  - шрифт, размер, жирность, цвет, прозрачность
  - выравнивание, межстрочный интервал, трекинг
  - тень, обводка, фоновая заливка текста
- Управление слоями: порядок, блокировка, скрытие, дублирование, копирование/вставка.
- Быстрые инструменты для текста:
  - выравнивание по безопасной зоне (лево/центр/право, верх/центр/низ)
  - подгонка текста по ширине слоя
  - умный контраст (авто цвет текста + обводка)
- Горячие клавиши для быстрого редактирования (`Cmd/Ctrl+Z`, `Cmd/Ctrl+Y`, `Delete` и др.).
- Автосохранение в `localStorage`.
- Сохранение и загрузка проектов (`.json`).
- Экспорт в PNG/JPG.
- Переключение интерфейса RU/EN.
- **Наложение безопасных зон** для учёта перекрытий UI соцсетей (особенно актуально для сторис).
- Мобильный флоу редактирования (нижние действия + боковые шторки для слоёв и стилей).
- Пастельная тема в минт-зелёных тонах + фирменный логотип.


## Деплой на GitHub Pages

В репозитории уже есть готовый воркфлоу: [`.github/workflows/deploy-pages.yml`](./.github/workflows/deploy-pages.yml)

1. Запушить в `main`.
2. В настройках репозитория GitHub включить **Pages** с источником **GitHub Actions**.
3. Дождаться завершения воркфлоу `Deploy to GitHub Pages`.

Воркфлоу собирает все пакеты монорепозитория и деплоит `apps/web/dist`.

## Архитектура монорепозитория

```text
apps/
  web/       React + Vite фронтенд редактор
  api/       опциональный бэкенд-пакет (песочница)
packages/
  core/      доменные типы, пресеты, фабрики
  editor/    стор состояния + история команд + адаптер Fabric
  ui/        переиспользуемые React-компоненты + тема + шрифты
  utils/     общие утилиты
```

## Позиционирование

MINT намеренно прост:

- нет обязательной регистрации
- нет сложных таймлайнов и движков шаблонов
- быстрое исполнение для тех, кто хочет опубликовать прямо сейчас
